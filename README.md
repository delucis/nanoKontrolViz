# nanoKontrolViz

__nanoKontrolViz__ is a Max “Extra,” which provides visualisation of incoming
MIDI messages from a Korg nanoKontrol 2.

Download to `~/Documents/Max 7/Packages/` (or `~/Documents/Max 7/Packages/` in
Max 6) to install.
